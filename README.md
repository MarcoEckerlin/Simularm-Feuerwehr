# 🚒 Simularm Feuerwehr 🚨

**Einsatzleitsystem für Feuerwehr-Simulationen – powered by Spring Boot + Thymeleaf**

> 💥 **Version:** 1.0

---

## 📦 Features

✅ Anzeige kommender Alarme  
✅ Durchsagen starten  
✅ Webfrontend mit Spring Boot & Thymeleaf  
✅ Einfach zu deployen mit `.bat`-Startdatei

---

## 🛠️ Installation

### 1️⃣ Java installieren ☕

- Gehe zu [https://www.oracle.com/java/technologies/javase-downloads.html](https://www.oracle.com/java/technologies/javase-downloads.html)  
- Lade **Java 21 (oder höher)** runter  
- Installiere es und prüfe in der Konsole:

```bash
java --version
```

---

### 2️⃣ Release herunterladen 📥

- Gehe zu den **Releases** auf GitHub und lade die fertige `.jar`-Datei herunter (als Asset angehängt)  
- Speichere sie in einen Ordner deiner Wahl

---

### 3️⃣ Starten mit .bat 🏁

- Schaue in deinem Projektordner nach der Datei: `start.bat`  
- Doppelklick → **Fertig!** 🎉

Oder per Konsole:

```bash
java -jar Simularm-Feuerwehr-1.0.jar
```

---

## 🌐 Anwendung öffnen

Öffne im Browser:

```
http://localhost:80 <- den port deiner Wahl ( mit --port= festgelegt)
```

👀 Schau dir dein Feuerwehr-Dashboard an!

---

## 💡 Nützliche Commands

| Task                 | Command                                        |
|----------------------|------------------------------------------------|
| Run Jar direkt      | `java -jar Simularm-Feuerwehr-1.0.jar`        |
| Mit Port und GUI    | `java -jar Simularm-Feuerwehr-1.0.jar --port=80` |
